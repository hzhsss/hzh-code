 ## 便利签背单词

这种背单词的方法，是自备考以来一直使用的。用便利签的颜色刺激视觉感官，让背单词的过程不这么枯燥无味。参考各类单词词汇书...

虽然知道这个主题的功能不是这么用的 - 

```md
::: tip
 原本作提示信息使用
 绿色标签
:::

::: warning 
 原本作注意标签使用
 黄色标签
:::

::: danger
 原本作警告标签使用
 红色标签
:::

::: note
 原本作笔记标签使用
 蓝色标签
:::
```

**效果**

::: tip 绿色

提示标签

:::

::: warning 黄色

注意标签

:::

::: danger  红色

警告标签

:::

::: note 蓝色

笔记标签

:::
